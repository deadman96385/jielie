## Pin usage

|    Pin    |             Usage              |
|-----------|--------------------------------|
| PR1       | Power button                   |
| PR2       | Amplifier shutdown             |
| PA0       | Microphone                     |
| PA3       | AUX in                         |
| PA4       | Keys1                          |
| PB0       | Onboard LEDs (BLUE1 and BLUE2) |
| PB4       | Keys2                          |
| PB5       | Keys3                          |
| PC5       | SD card CLK                    |
| PC4       | SD card CMD                    |
| USBDM/PC3 | SD card DAT0, USB D-           |
| USBDP     | USB D+                         |
