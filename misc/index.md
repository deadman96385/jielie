# Misc stuff

- [USB IDs](usb-ids.md) - List of the USB IDs that is used by JieLi chips and stuff based on them.
- [Chip IDs](chipid.md) - List of the chip id values stored in the `JL_SYSTEM->CHIP_ID` register.
- [USB_KEY](usb-key.md) - The signal used to enter the USB bootloader
- [UART_KEY](uart-key.md) - The signal used to enter the UART bootloader
- [ISP_KEY](isp-key.md) - Alternative way of entering the USB bootloader in case the USB_KEY hasn't been a viable option.
